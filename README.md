# BARD

## Data
Dataset combination of:
* Berkeley DeepDrive 100K (BDD100K)
* Adverse Conditions Dataset with Correspondences (ACDC)
* Real-world Task-driven Testing Set (RTTS)
* Detection in Adverse Weather Nature (DAWN)
